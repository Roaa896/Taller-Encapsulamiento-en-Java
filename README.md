*INTRODUCCIÓN:*    
En este taller se estudia uno de los principios fundamentales de la Programación Orientada a Objetos en Java: el encapsulamiento. Este concepto permite proteger los datos de una clase mediante el uso de modificadores de acceso y métodos especiales como getters y setters, garantizando así un control adecuado sobre la información. A lo largo de los ejercicios, se busca comprender cómo ocultar los atributos internos de una clase y cómo interactuar con ellos de forma segura. El objetivo es desarrollar habilidades para crear programas más seguros, organizados y mantenibles, sentando las bases para un mejor diseño de sistemas orientados a objetos.
